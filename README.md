#  Flight Booking Automation Framework

##  Overview

This project is a Selenium-based Automation Testing Framework built for the BlazeDemo flight booking application.

---


##  Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* WebDriverManager
* Extent Reports

---

##  Key Features

*  Page Object Model (POM)
*  DataProvider for multiple test inputs
*  Configurable browser & environment
*  Headless execution support
*  Screenshot capture on failure
*  HTML reporting with Extent Reports

---


This project demonstrates a **complete end-to-end automation framework** aligned with real-world QA practices.
